About

BackOnTrack is an app designed to help you stay organized. You can create tasks, set start and finish dates for them, add tags, set task's priority. Later, you can sort your tasks by dates, priority or filter them by the tags you've created. You can also browse the notes using a weekly view which shows notes with set dates.
